COMPUTER VISION

Bij dit onderdeel wordt er kennis gemaakt met Processing. Door middel van het maken van opdrachten ga je kennis en ervaring opdoen. De opdrachten komen uit de reader "HCI Technologies 2019 - Processing & Computer Vision". Processing is een open source programmeertaal en een omgeving voor mensen die interactieve afbeeldingen en animaties willen programmeren. De programmeertaal is vergelijkbaar met Java, maar makkelijker om te gebruiken.

Opdracht 1

De voorwaarden voor opdracht 1 zijn:

  • Minimaal drie ellipsen
  • Ellipsen op verschillende locaties
  • Elipsen met verschillende kleuren
  • Verschillende breedtes en hoogtes
  • Met en zonder randen

Ik heb drie ellipsen geprogrammerd. Deze ellipsen staan op verschillende locaties, hebben verschillende kleuren, verschillen van breedtes en hoogtes en hebben een rand.


Opdracht 2

De voorwaarden voor opdracht 2 zijn:

  • Minimaal vier verschillende vormen (geen cirkels)
  • De vormen in verschillende kleuren
  • De vormen op verschillende locaties

Ik heb een witte lijn, een paarse driehoek met bruine randen, een fel licht blauwe vierkant met gele randen en een roze-paars quad met blauwe randen geprogrammeerd. Ze staan allemaal op verschillende locaties.


Opdracht 3

De voorwaarden voor opdracht 3 zijn:

  • Een programma dat tenminste twee verschillende vormen
  • De vormen moeten verschillende kleuren hebben
  • Het programma moet interactief zijn

Ik heb een cirkel geprogrammeerd die de muis volgt. De cirkel wordt ondertussen groter en verandert van kleur. De lijn volgt de muis ook mee.


Opdracht CV1

Voor opdracht CV1 moet je de code van LiveCamTest openen. Er wordt gebruik gemaakt van de webcam. Door de code herkent de webcam een gezicht en zet er een vierkant rondom het gezicht. Je moet de code aanpassen zodat er een cirkel rondom je hoofd zit i.p.v. een vierkant.


Opdracht CV2

Voor opdracht CV2 moet je de code van LiveCamTest aanpassen zodat elk herkent gezicht anoniem wordt. Dit kan je doen door middel van een zwarte balk voor de ogen of een rode colwns neus. Ik heb voor het laatste gekozen.


Bonus opdracht

Voor de bonus opdracht moet je weer de code van LiveCamTest aanpassen. Je moet een bril of een hoed bij het herkende gezicht geven. Ik heb gekozen voor een bril.


Opdracht CV3

Voor opdracht CV3 moet je de code OpenCVMultipleColorTrackingLiveCam en OpenCVBrightestPointDetectionLiveCam downloaden en kijken wat ze doen. Daarna ga je bij een van deze codes de code aanpassen zodat er iets cools gebeurd op het scherm. Ik heb gekozen voor de code OpenCVBrightestPointDetectionLiveCam en de filter THRESHOLD aan toegevoegd. Hierdoor wordt het beeld van de webcam nu zwart en wit en het contrast is heel sterk. Daarnaast zijn de lijnen veel boller geworden.


Opdracht Augmented Reality

Voor de opdracht AR ga je augmented markers voor de camera plaatsen en kijken wat er gebeurd. Op de afbeelding zie je dat er een 3D kubus op de plaats van de marker verschijnt.
